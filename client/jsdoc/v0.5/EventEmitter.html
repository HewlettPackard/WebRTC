<static> off(types, listener)

Un-bind/un-register a callback function from one or more event types.

This function un-binds (i.e., un-registers) a callback function from one or more event types.

Parameters:
Name Type Description
types string

A string of one or more event types (whitespace separated) the listener is to be un-registered from:

  • signaling : signaling channel events.
  • fsm : fsm events (call).
  • ice : ICE events.
  • call : call events.
  • session : session events.
  • media : media events.
  • event : 'generic' events.
  • error : error and timeout events.
listener function

The callback function provided when the listener was bound.
